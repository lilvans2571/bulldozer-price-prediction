# Bulldozer Price Prediction

This project builds a machine learning model to predict bulldozer auction sale
prices using historical sales data and machine features.

## Evaluation Metric
RMSLE was used as the primary evaluation metric.

## Model
- Baseline: RandomForestRegressor
- Tuned using RandomizedSearchCV

## Contents
- `bulldozer_analysis.ipynb`: full ML workflow
- `report.md`: project summary
- `requirements.txt`: dependencies
